# Whozin Brand Handoff Package

This package is for design and engineering handoff. It includes:

- logo direction based on the current product mark
- reusable color and radius tokens
- CSS custom properties for quick implementation
- JSON tokens for design systems, Figma variables, or frontend mapping

Files:

- `logo-spec.md`
- `brand-tokens.json`
- `brand-tokens.css`

Important note:

Whozin now ships with a formalized in-product mark in the repo: a rounded-square gradient container with a white `W` glyph. This package documents that current state so design and engineering can keep the mark consistent across product, PWA, and marketing surfaces until a fuller brand system is designed.
