# Whozin Brand Handoff Package

This package is for design and engineering handoff. It includes:

- logo direction based on the current product mark
- reusable color and radius tokens
- CSS custom properties for quick implementation
- JSON tokens for design systems, Figma variables, or frontend mapping

Files:

- `logo-spec.md`
- `brand-tokens.json`
- `brand-tokens.css`

Important note:

Whozin does not yet have a formal exported logo asset in the repo. The current logo direction is inferred from the live welcome experience: a rounded-square gradient container with a white `Zap` glyph. This package documents that current state so the team can use it consistently until a final mark is designed.
